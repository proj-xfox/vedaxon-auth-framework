# Auth Framework

A **reusable authentication framework** for Node.js applications. Provides utilities for JWT authentication, password hashing, session creation, middleware, and custom error handling. Ideal for building secure REST APIs or any Node.js service requiring user authentication.

---

## Features

- Generate and verify JWT tokens
- Password hashing and comparison using bcrypt
- Express.js middleware for token validation
- Create standardized session objects
- Custom `AuthError` class for consistent error handling
- Configurable via environment variables or programmatically

---

npm install auth-framework
